# React Learning Journey 🚀

Is repository mein maine React aur frontend development seekhte hue jo projects aur practice files banayi hain, unka collection hai. Vite + React ka use karke component-based development, hooks, state management, aur styling (CSS/Bootstrap) practice ki gayi hai.

## 🛠️ Tech Stack

- **React** (with Vite)
- **JavaScript (ES6+)**
- **CSS3 / Bootstrap 5**
- **HTML5**
- **Git & GitHub** (version control)

## 📂 Projects

### 1. [Todo App](./Projects/1-todo-app-version-1)
Ek simple Todo List application jisme tasks add/manage kiye ja sakte hain. Component structure aur props ka practice.

**Key concepts:** Component composition, props passing, conditional rendering

---

### 2. [Bharat Clock](./Projects/2-bharat-clock-version-one)
Ek real-time digital clock application jo current time show karta hai.

**Key concepts:** `useState`, `useEffect`, `setInterval` for live updates

---

### 3. [Bootstrap Integration](./17Bootstrap)
Bootstrap 5 ko React project ke saath integrate karne ki practice — responsive components banane ke liye.

**Key concepts:** Bootstrap classes with JSX (`className`), CSS specificity handling

---

### 4. [Fragments & Component Practice](./18Fragments/learningPractice)
React Fragments aur reusable component patterns (FoodItems, Item, ErrorMessage) ki practice.

**Key concepts:** React Fragments, component reusability, error handling UI

---

### 5. [First Component Practice](./create-first-comp)
React seekhne ki shuruaat — pehla component banana aur basic JSX syntax samajhna.

**Key concepts:** JSX basics, functional components, exporting/importing components

## ⚙️ How to Run Any Project Locally

Har project folder ke andar jaake ye commands chalayein:

```bash
# Dependencies install karein
npm install

# Development server start karein
npm run dev
```

Fir browser mein terminal pe diya gaya localhost link (usually `http://localhost:5173`) open karein.

## 📌 Note

- Har project ek independent Vite + React setup hai
- `node_modules` folder ko `.gitignore` mein rakha gaya hai (ise install karke generate karein)
